
# Smart Parking Occupancy Analytics

## Overview

This project analyzes parking occupancy data to identify peak usage periods, parking zone utilization, vehicle distribution, and average parking duration. The insights help improve parking allocation and reduce congestion through data-driven decision-making.


## Objectives

- Analyze parking occupancy trends
- Identify the busiest parking zones and time slots
- Analyze vehicle type distribution
- Calculate average parking duration
- Generate business insights and recommendations
## Technologies Used  

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook / Google Colab
## Dataset Information

The dataset contains the following fields:
- Date
- Day
- Time Slot
- Parking Zone
- Total Parking Spaces
- Occupied Spaces
- Available Spaces
- Vehicle Type
- Average Parking Duration
## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Parking Occupancy by Zone
- Peak Parking Time Analysis
- Vehicle Type Analysis
- Average Parking Duration
- Business Insights & Recommendations

## Skills Covered
- Data Cleaning
- Data Analysis
- GroupBy Operations
- Data Visualization
- Business Insights
## Project Questions

1. Which parking zone is occupied the most? 
2. At what time is parking busiest? 
3. Which vehicle type occupies more parking spaces?
4. What is the average parking duration? 
5. How can parking congestion be reduced?
